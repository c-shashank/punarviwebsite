# Punarvi Website

## Project Structure

```
PurnaviWebsite/
├── index.html              # Main homepage
├── assets/
│   ├── css/
│   │   └── main.css       # Main stylesheet
│   ├── js/
│   │   └── main.js        # Main JavaScript file
│   └── images/            # Image assets
├── products/               # Product pages
├── blog/                   # Blog pages
└── data/                   # Data files (JSON, etc.)
```

## Getting Started

1. Open `index.html` in your browser
2. Start editing files to customize your website

## File Structure

- **HTML**: `index.html` - Main page structure
- **CSS**: `assets/css/main.css` - All styling
- **JS**: `assets/js/main.js` - JavaScript functionality
- **Images**: `assets/images/` - Place all images here
